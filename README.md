# amall
仿github上star5.2k的大型商城项目前端部分

### 所用技术
- vue
- axios
- element-ui
- mockjs
- vue-router
- vuex
- sass
- vue-lazyload

### 效果
- 页面内所有数据均由mock.js模拟产生
